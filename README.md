## 背景
毕业纪念册记录着同学们校园时光的点点滴滴，是同学们友情、爱情最好的见证。但现如今市面上的毕业纪念册就两种，一种是纸质版的毕业纪念册，另一种是通过H5，功能单一，这两种就是属于仅为照片记录，且易丢失。目前市面上没有可以实时更新纪念册同学们的信息的纪念册，因此市场竞争压力小，且空间潜力大。

## 目标人群
